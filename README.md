# WordMerger

**WordMerger** — это настольное приложение для объединения нескольких документов Microsoft Word (`.docx`) из одной папки в один итоговый файл.  
Приложение позволяет гибко управлять порядком объединения документов, поддерживает сортировку по имени, дате создания и по регулярному выражению, а также корректно работает с изображениями, таблицами и стилями Word.

Каждый исходный документ в результирующем файле начинается **с новой страницы**.

## Установка зависимостей

Рекомендуется использовать виртуальное окружение.

```bash
python -m venv .venv
```

### Активация окружения

**Windows**

```bat
.venv\Scripts\activate
```

**Linux / macOS**

```bash
source .venv/bin/activate
```

### Установка зависимостей

```bash
pip install -r requirements.txt
```

## Запуск приложения в режиме разработки

После установки зависимостей приложение можно запустить напрямую:

```bash
python app/main.py
```

Откроется графическое окно приложения.

## Сборка в exe-файл (Windows)

Для создания самостоятельного `.exe` файла используется скрипт `build.py`.

### Сборка

```bash
python build.py
```

После успешной сборки файл будет доступен по пути:

```
dist/WordMerger.exe
```

Установка Python на компьютере пользователя в этом случае **не требуется**.

## Как пользоваться приложением

1. Запустите приложение (`WordMerger.exe` или `python app/main.py`).
2. Нажмите **«Выбрать папку»** и укажите директорию с файлами `.docx`.
3. Выберите способ сортировки:

   * по имени
   * по дате создания
   * случайный порядок
   * по регулярному выражению
4. При выборе сортировки по регулярному выражению:

   * задайте шаблон имени файла
   * используется первая захватывающая группа `(...)`
   * пример для файлов `1_name.docx`, `2_name.docx`:

     ```
     ^(\d+)_.*\.docx$
     ```
5. Выберите порядок сортировки:

   * по возрастанию
   * по убыванию
6. Нажмите **«Объединить»**.
7. После успешного объединения нажмите **«Выгрузить»** и укажите место сохранения итогового файла.
